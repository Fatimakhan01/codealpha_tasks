# CodeAlpha Full Stack Tasks

This repository contains the completed internship tasks for **CodeAlpha**, developed using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.

The purpose of these tasks is to demonstrate practical full-stack development skills, including frontend development with React, backend API development with Node.js & Express, and database integration using MongoDB Atlas.

---

## Task 1: MERN E-Commerce Application

A full-stack e-commerce web application built using the MERN stack.

### Key Features
- User authentication (signup & login)
- Product listing from MongoDB database
- Add to cart and remove from cart functionality
- Order placement and order storage
- User order history
- Responsive user interface

### Tech Stack
- **Frontend:** React.js, Context API, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT & bcrypt
- **API Testing:** Postman

---

## Task 2: Mini Social Media Application

A social media platform developed using the MERN stack that allows users to interact with posts and other users.

### Key Features
- User registration and authentication
- Create, update and delete posts
- Like and comment on posts
- User profiles
- Follow / user-based content handling
- Secure APIs with authentication middleware

### Tech Stack
- **Frontend:** React.js, Context API, Axios
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Authentication:** JWT
- **API Testing:** Postman

---

## Common Technologies Used
- MERN Stack (MongoDB, Express.js, React.js, Node.js)
- RESTful APIs
- JWT Authentication
- MongoDB Atlas (Cloud Database)
- Git & GitHub for version control
